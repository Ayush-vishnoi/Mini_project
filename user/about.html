About This Project

Welcome to the Self-Ordering Restaurant platform — a user-friendly food ordering website built with HTML, CSS, and JavaScript. This project allows customers to browse menus, select items with quantity controls, add them to their cart, and complete an order with a simple checkout system.

It showcases skills like front-end web development, DOM manipulation, dynamic cart updating, local storage management, and responsive design principles. Additional features like real-time order updates, quantity adjustment, and SweetAlert2-based confirmation popups have been incorporated for a smooth user experience.
